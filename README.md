# 招聘大数据可视化分析系统 python223

## 项目概述

招聘大数据可视化分析系统是一款基于Python开发的Web应用，主要用于分析招聘信息，实现数据可视化，并提供岗位匹配与薪资预测等智能功能。

## 技术栈

- 前端：Flask
- 后端：MySQL

## 功能模块

### 1. 首页数据展示

- 岗位信息数量
- 行业平均薪资、最低薪资与最高薪资
- 学历与薪资统计
- 整体学历要求
- 福利待遇
- 热门岗位
- 数据可视化

### 2. 数据概况

- 开发岗位分类与非开发岗位分类
- 薪资预测与岗位匹配
  - 支持学历要求搜索
  - 支持职位输入搜索

### 3. 可视化分析

#### 薪资情况

- 支持根据学历查看岗位薪资
- 使用柱状图、饼图展示薪资分布与比例

#### 企业情况

- 查看职位招聘主要城市
- 查看公司规模
- 展示职位在各主要城市的比例

#### 学历情况

- 展示职位对学历与工作经验要求
- 使用漏斗图、矩形树图进行可视化

### 4. 智能化服务

#### 薪资预测

- 支持多条件选择（职位、学历、公司规模、城市、工作经验）
- 支持不同模型预测最低薪资

#### 职位匹配

- 支持多条件筛选
- 筛选最符合条件职位信息

## 系统角色

- 用户：查看分析数据，使用薪资预测和职位匹配功能
